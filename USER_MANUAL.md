# User Manual

This manual directs site engineers and analysts to the resources needed for installation, configuration, and advanced operation of the PowerMonitor 500 integration concept.

## Installation Workflow
1. Review prerequisites and detailed setup steps in [`docs/user/install_guide.md`](docs/user/install_guide.md).
2. Configure device assemblies and transport modes using the reference in [`docs/user/configuration.md`](docs/user/configuration.md).
3. Apply polling and connection guardrails from [`docs/user/advanced_usage.md`](docs/user/advanced_usage.md) before onboarding additional consumers.
