# Example Assets

Sample configurations and datasets referenced by the documentation live here. They provide quick-start material for lab explorations and automated tests.

## Contents

- `telemetry-sample.json` – Mock payload that mirrors the Node-RED assembly export documented in `docs/telemetry-pipeline.md`.
- `influxdb/bootstrap.env.example` – Template environment variables for provisioning the InfluxDB stack described in `src/infrastructure/influxdb/`.
